E-Commerce Backend Assignment
This is a secure Node.js backend built for smooth data management and product insights.
Core Features
Auth: Secure Login and Registration using JWT and bcryptjs.
Security: Role-based access (Admin, User, Guest).
Products: CRUD operations with search and category filtering.
Analytics: Product recommendation system integration.
Clean Code: Modular folder structure (Routes, Controllers, Models).
Folder Guide
config/: Database connection.
controllers/: Logic for auth, products, and orders.
models/: Schemas for Users, Products, and Orders.
routes/: API endpoints
middleware/: Security checks and error handling.
Setup
Run npm install to install dependencies.
Add your MongoDB link to the .env file.
Run npm run dev to start the server.
